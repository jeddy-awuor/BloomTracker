# BloomTrack — Personal Productivity & Project Tracker

A beautiful, girly pink personal productivity app built with Next.js (App Router), React, TypeScript, and Tailwind CSS.

## Features

- **Dashboard** — Overview of projects and weekly task progress
- **Projects** — Create projects, track tasks, and see visual progress bars
- **Weekly Tasks** — General weekly checklist with completion summary
- **Import Text** — Paste plans and turn them into tasks
- **Supabase sync** — Sign in to sync data across devices (optional)
- **LocalStorage fallback** — Works offline without Supabase configured

## Getting Started

```bash
npm install
npm run dev
```

## Tech Stack

- Next.js 15 (App Router)
- React 19
- TypeScript
- Tailwind CSS
- Lucide React icons
- Supabase (auth + PostgreSQL)
